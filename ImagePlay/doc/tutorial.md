Getting started {#gettingstarted}
==========================

Installing (Windows)
--------------------------
1. Run setup.exe
2. Run ImagePlay.exe

Installing (Mac OS X)
--------------------------
1. Mount ImagePlay.dmg
2. Drag & Drop to /Applications/
3. Run ImagePlay.app
